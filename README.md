# Cyber-Nexus // Cognitive Security Operations Center

Cyber-Nexus is a high-tech, futuristic cybersecurity dashboard representing a Cognitive Security Operations Center (CSOC). It is built with React, Vite, HTML5 Canvas, SVG graphs, Vanilla CSS, and native Web Audio API oscillators.

## 🚀 Key Features

- **🌐 Global Threat Vector Map**: Real-time canvas attack lines traveling between world hacker hotspots and defensive nodes with glowing particle effects and threat logs.
- **💻 Interactive Hacker Terminal**: A fully functional CRT terminal supporting command diagnostics:
  - `scan [ip]`: sweeps network nodes with ascending Web Audio frequencies
  - `decrypt`: fallout-style hex word-guessing game (access bypass challenge)
  - `system`: mainframe and RAM load analysis
  - `logs`: scrollable firewall events interceptor
  - `matrix`: full terminal green matrix code stream screensaver
- **🧠 Neural Node Visualizer**: Particle network graph reacting to user mouse connections. Allows operators to click and inject "Virus" (red propagation) or deploy "Patches" (blue containment).
- **📊 Real-time System Load HUD**: Dynamic SVG graph charts tracking CPU cores load and network throughputs using live mathematical random walk paths, combined with RAM percentage circles.
- **🎨 Global Multi-Theme Selector**: Toggle color custom properties across four themes (Cyan Cyber, Crimson Threat, Acid Matrix, and Obsidian Gold) affecting dashboard styles.
- **🔊 Synthesized Web Audio FX**: Retro cybernetic beep, alert siren, matrix chirp, and click sounds generated dynamically in-browser.

## 🛠️ Technology Stack

- **Framework**: React 19 + Vite 8
- **Styling**: Vanilla CSS (CSS Grid, backdrop-filter glassmorphism, @keyframe glitches, CRT filter scanlines)
- **Visuals**: HTML5 Canvas 2D + dynamic SVG Paths
- **Sound**: Native Web Audio API Context (Oscillators, Gains, and Frequency Sweeps)

## 📦 How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run local development environment:
   ```bash
   npm run dev
   ```

3. Build production bundle:
   ```bash
   npm run build
   ```
